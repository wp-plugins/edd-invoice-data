== Description ==

*English below*

EDD Invoice Data, to moduł do [Easy Digital Downloads](http://wordpress.org/extend/plugins/easy-digital-downloads/), który pozwala na łatwe zbieranie danych do faktury przy składaniu zamówienia. Pobieranie dodatkowych danych można aktywować dla dowolnej bramki płatności EDD.

= Możliwości =

* Pobieranie danych do faktury dla dowolnej bramki płatności EDD
* Podanie danych do faktury może być opcjonalne lub wymagane
* Pobranie danych do faktury od osoby fizycznej (imię, nazwisko i adres)
* Zebrane dane mogą być wykorzystane przez moduły integrujące z systemami fakturującymi ([WP Fakturownia](http://upsell.pl/sklep/wp-fakturownia/), [WP Taxe](http://upsell.pl/sklep/wp-taxe/), [WP iFirma](http://upsell.pl/sklep/wp-ifirma/), [WP wFirma](http://upsell.pl/sklep/wp-ifirma/))

*English version*

EDD Invoice Data is is add-on plugin for Easy Digital Downloads plugin, which allows you to easily gather invoice data for any EDD payment gateway during checkout.

= Features =

* Gathering the invoice data for any payment gateway
* Provide the invoice details can be optional or required
* Getting the invoice data from a individual (name and address)
* Dedicated mainly for EU entrepreneurs

== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `edd-invoice-data` and all its contents to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Downloads -> Settings -> Gateways and set your preferences.
